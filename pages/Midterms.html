    • Cascading Style Sheets

    Language used to specify the presentation aspects (e.g layout and formatting) of structurally marked up documents.
    Developed by Hakon Wium Lie (HTML Cascading Style Sheets /CHSS) and Bert Bos (Stream-based Sheet Proposal / SSP).

    1. CSS Level 1 (CSS 1, W3C Recommendation, Dec 1996)

    2. CSS Level 2 (CSS 2, W3C Recommendation, May 1998)

    3. CSS Level 2 Revision 1 (CSS 2.1, W3C Recommendation, June 2011)

    4. CSS Level 3 -modular approach to CSS development (as opposed to the monolithic specification of CSS 2.1)
